deploy42
=========
Read a YAML and export this as a 'Command as a Service' :p 

https://github.com/andrerocker/deploy42

##### Example

Based on a simple yaml
```yaml
daemon:
  bind: 127.0.0.1
  port: 8888
  http:
    pipe: true
    vars: true
  load:
    - /etc/deploy42/config.d/*.yml
    - /var/www/*/config/deploy42.yml

namespaces:
  - endpoint: admin-ops
    chaining:
      - a_audit_filter
      - a_security_filter
    commands:
      process:
        - get: ps -ef | grep {process}
          put: kill {process}
          delete: kill -9 {process}

  - endpoint: free-path
    commands:
      log:
        - get: tail -f {log}
        
      echo:
        - put: cat -
```

You can do this
